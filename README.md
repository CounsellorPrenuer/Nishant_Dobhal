# Nishant Dobhal Website

This repository contains the production Next.js website code for **Nishant Dobhal / Avyaan Learning**.

## Live Sanity Studio
- https://avyaan-learning-gc9hrjy5.sanity.studio/

## Website Sections
- Home
- About
- Services
- Packages
- Testimonials
- Contact

## Local Run
```bash
npm install
npm run dev
```

Open: http://localhost:3000

## Notes
- This repo contains website code only (root/main).
- Sanity Studio schema/deploy files are intentionally not kept in this repo as requested.
- Website content is pulled from the Sanity dataset configured in environment variables.
